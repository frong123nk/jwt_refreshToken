# jwt_refreshToken

#### Add .env file 
```
 ACCESS_SECRET = YOUR_ACCESS_SECRET_KEY
 REFRESH_SECRET = YOUR_REFRESH_SECRET_KEY
```
#### URL Paths

```
 /api/v1/login (login)
 /api/v1/logout (logout)
 /api/v1/refresh (get refresh access token)
 /api/v1/todo (test bearer token)
```

#### Start 

```
go run main.go
```
